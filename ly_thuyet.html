Ôn Tập kiến thức

  1. Khái niệm về git:
    Git là nột hệ thống điều khiển phiên bản (version control systerm) theo hình thức phân tán. Được dùng để ghi nhận mã nguồn (source code) và ghi lại các thay đổi.

  2. Các lệnh cơ bản của Git

    • Git init: Khởi tạo một Repository
    • Git add: Đưa các file vào vùng staging
    • Git commit: Ghi nhận các thay đổi
    • Git remote add origin: Kết nối local repo với remote repo
    • Git push: Đưa các thay đổi từ local repository lên remote repo

  3. Khái niệm html:
